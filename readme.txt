== Installation ==

**Quick Start Guide:**

1. **Upload & Activate**
   * Upload the `heat-map-graph` folder to `/wp-content/plugins/`
   * Activate the plugin through the 'Plugins' menu in WordPress

2. **Create Your First Chart**
   * Navigate to "Heat Map Graph" in your WordPress admin menu
   * Click "Add New Graph" or explore the sample graphs
   * Configure your SQL query, field mappings, and color scheme
   * Save and preview your visualization

3. **Display Your Chart**
   * Copy the shortcode: `[heat_map_graph id="123"]`
   * Paste it into any page, post, or widget where you want the chart to appear
   * Your chart will render automatically!

**Pro Tip:** Start with the sample graphs to see how everything works, then customize them for your needs!

= Which tables can I query? =
You can query any WordPress core tables prefixed by your site's `$wpdb->prefix`. This includes:
* `wp_posts`, `wp_users`, `wp_comments`
* `wp_postmeta`, `wp_usermeta`, `wp_commentmeta`
* `wp_terms`, `wp_term_taxonomy`, `wp_term_relationships`
* And any other WordPress core tables

= Can I use external data sources? =
Yes! Premium users can connect to:
* REST APIs (JSON endpoints)
* CSV files hosted online
* Any publicly accessible data source

This allows you to visualize data from external systems, government APIs, third-party services, or your own custom APIs.

= Can I pass parameters to queries? =
For security reasons, dynamic user input is not supported in SQL queries. Use static queries or database views. However, Premium users can leverage external APIs that accept parameters, giving you dynamic data visualization capabilities.
